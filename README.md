# Cybersec-task-7
***Identify and Remove Suspicious Browser Extensions***

**Objective:**
The objective of this task is to review installed browser extensions, identify any potentially harmful or unnecessary extensions, and remove them to improve browser security and performance. 

**Tools Used:**
Google Chrome (Windows)  Built-in Extension Manager (chrome://extensions/) 
Step 1: Open Extension Manager  Open Google Chrome.  Click on the three dots (⋮) in the top-right corner.  
Navigate to Extensions → Manage Extensions OR type the following in the address bar:  chrome://extensions/  

**Step 2: Review Installed Extensions:**
I carefully reviewed all installed extensions. 
Currently installed extension:  Google Docs Offline (Official extension by Google)  
I verified:  Developer name  Number of users  Reviews and ratings Last updated date  The extension is published by Google and has millions of users, indicating it is legitimate and safe. 

**Step 3: Check Permissions:**
I checked the permissions granted to the extension.  
The Google Docs Offline extension requires:  Access to Google Docs files for offline editing  Limited browsing interaction related to Google services  The permissions were appropriate for its intended functionality.

**Step 4: Identify Suspicious or Unused Extensions  After reviewing all installed extensions:** 
No suspicious extensions were found. 
No unknown or unused extensions were present. 
No extensions requested excessive or unrelated permissions.  

**Step 5: Action Taken  Since no suspicious or unnecessary extensions were found:**  
No extensions were removed.  
Browser settings remained unchanged. 
No performance issues were observed. 

**Step 6: Research – What If Suspicious Extensions Were Found?** 
If a suspicious extension had been detected, the following actions would have been taken:  
Verify developer credibility and reviews. 
Check permissions for unnecessary access.  
Remove the extension immediately.  
Restart the browser.  
Clear browsing data if needed.  
Scan system using antivirus software.  
Malicious extensions can:  
Steal personal information  Capture login credentials  Inject unwanted advertisements  Redirect search engines  Track browsing activity  Therefore, regularly reviewing extensions is an important security practice.

**Outcome**

Gained awareness of browser extension security risks.
Learned how to review extension permissions.
Verified installed extensions for legitimacy.
Understood preventive measures against malicious extensions practice practice.
